# Team Task Manager

A full-stack task management application where teams can create, assign, and manage tasks.

## Features
- User authentication
- Create tasks
- Assign tasks
- Update task status
- Team collaboration

## Tech Stack
Frontend:
- React
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB

## Installation

### Frontend
```bash
cd client
npm install
npm start
